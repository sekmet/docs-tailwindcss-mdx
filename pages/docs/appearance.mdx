## Usage

Use `.appearance-none` to reset any browser specific styling on an element. This utility is often used when creating [custom Form components](/docs/examples/forms).

@component('_partials.code-sample', ['class' => 'text-center'])
<div className="max-w-sm mx-auto">
  <div className="flex my-4">
    <select className="w-16">
      <option>Yes</option>
      <option>No</option>
      <option>Maybe</option>
    </select>
    <div className="mx-4">
      Default browser styles applied
    </div>
  </div>
  <div className="flex my-4">
    <select className="appearance-none w-16">
      <option>Yes</option>
      <option>No</option>
      <option>Maybe</option>
    </select>
    <div className="mx-4">
      Default styles removed
    </div>
  </div>
</div>
@slot('code')
<select>
  <option>Yes</option>
  <option>No</option>
  <option>Maybe</option>
</select>

<select className="appearance-none">
  <option>Yes</option>
  <option>No</option>
  <option>Maybe</option>
</select>
@endslot
@endcomponent
